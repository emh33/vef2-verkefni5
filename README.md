# Vefforritun 2 - Verkefni 5

## Keyra verkefni á eigin vél

```bash
git clone git@github.com:emh33/vef2-verkefni5.git
cd vef2-verkefni5
npm install
npm run lint
npm run build
npm start
```

Þá keyrir síðan á http://localhost:3000/
